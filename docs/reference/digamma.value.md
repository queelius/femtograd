# Digamma (psi) function for value objects

Computes the digamma function psi(x) = d/dx log(gamma(x)). This is the
derivative of lgamma and appears in gradients of many statistical
distributions.

## Usage

``` r
# S3 method for class 'value'
digamma(x)
```

## Arguments

- x:

  A value object

## Value

A new value object representing digamma(x)
