# Print value object and its computational graph

Print value object and its computational graph

## Usage

``` r
# S3 method for class 'value'
print(v, depth = Inf, indent = "  ", ...)
```

## Arguments

- v:

  A value object

- depth:

  Integer indicates depth (dfs) to recurse down the graph

- indent:

  A character string specifying the indentation for each level in the
  computational graph (default: " ")
